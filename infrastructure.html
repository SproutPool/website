Server Infrastructure

Overview

Description

Our infrastructure is set up in a way to provide a high level of redundancy and maximum security. Here are some key features of our setup:

  • 2 availability zones for nodes: If Amazon experiences issues with one, we can easily leverage the other and continue producing blocks.
  • Backup producer: In case there are ANY issues with the main producer node, we can easily switch over to our backup producer.
  • Highly specific security groups: Security groups limiting inbound traffic to only to the necessary connections.
  • Bastion host: There to act as a proxy server to protect our servers from any sort of cyber attacks.
